Modeling a single waiting time

At a drive-thru restaurant, customers spend, on average, 3 minutes placing their orders […] find the probability that customers spend less than 6 minutes placing their order.

– (Strang and Herman 2016, chap. 5.2)

We’ll refer to this process of placing one’s order as a random variable, \(X\), modeled using an exponential probability density function (pdf): \[ f(x) = \begin{cases} \frac{1}{\beta} e^{-\frac{x}{\beta}} \quad &x \geq 0 \\ 0 \quad &x < 0 \end{cases} \tag{1}\] Where \(\beta\) is the average waiting time.

We set the equation up in R so that we can inspect a plot and better understand the problem:

P_x <- function(x, beta) {
  case_when(x < 0 ~ 0,
            x >= 0 ~ (1/beta) * exp(-x/beta))
}
Code 
plot_ly(
  x = x_vec,
  y = P_x(x_vec, 3),
  type = "scatter",
  mode="line",
  name = "P(x)") %>% add_trace(
    x = 0:6,
    y = P_x(0:6, 3),
    type = "scatter",
    mode="none",
    fill = "tozeroy",
    name = "P(x) <= 6"
)

\[ \begin{align*} P(X \leq 6) &= \int_0 ^6 f(x)dx \\ &= \int_0 ^6 \frac{1}{\beta} e^{-\frac{x}{\beta}}dx \\ \text{Substitute}\, u &= -\frac{x}{\beta} \\ \frac{du}{dx} &= -\frac{1}{\beta} \rightarrow du = -\frac{1}{\beta}dx \\ P(X \leq 6) &= \int_0 ^6 \frac{1}{\beta} e^udx \\ &= \int_0 ^6 e^u \frac{1}{\beta}dx \\ \text{Substitute}\, du &= -\frac{1}{\beta}dx \\ P(X \leq 6) &= \int_0 ^6 e^u (-du) \\ &= -1\int_0 ^6 e^u du \\ &= - e^u \Big|_0^6 \\ P(X \leq 6) &= - e^{-\frac{x}{\beta}} \Big|_0^6 \\ \end{align*} \]

\(- e^{-\frac{x}{\beta}} \Big|_0^6 = - e^{-\frac{6}{3}} - - e^{-\frac{0}{3}} = 1 - e^{-\frac{6}{3}}\)

(-exp(-6/3)) + 1
[1] 0.8646647

When we derived the indefinite integral in order to set up our definite integral problem, we derived another function – the cumulative density function (cdf)

\[ F_X(x) = P(X \leq x) = \int_0^xf_X(t)dt \]

where \(f_X(t)\) is the probability density function. In our case, we can define the cumulative density function of the exponential distribution as follows:

cdf_x <- function(x, beta){
  -1 * exp(-x/beta)
}

and confirm that \(F_X(x) = e^{-\frac{x}{\beta}} \Big|_0^6\)

1 + cdf_x(6, 3)
[1] 0.8646647
plot_ly(
  x = x_vec,
  y = 1 + cdf_x(x_vec, 3),
  type = "scatter",
  mode="line",
  name = "F(X, Y)")

### Modeling a single waiting time

> At a drive-thru restaurant, customers spend, on average, 3 minutes
> placing their orders \[...\] find the probability that customers spend
> less than 6 minutes placing their order.
>
> -- [@strang_calculus_2016-1, chapter 5.2]

We'll refer to this process of placing one's order as a random variable,
$X$, modeled using an exponential probability density function (pdf): $$
f(x) = \begin{cases}
\frac{1}{\beta} e^{-\frac{x}{\beta}} \quad &x \geq 0 \\
0 \quad &x < 0 
\end{cases}
$$ {#eq-all} Where $\beta$ is the average waiting time.

We set the equation up in `R` so that we can inspect a plot and better
understand the problem:

```{r}
P_x <- function(x, beta) {
  case_when(x < 0 ~ 0,
            x >= 0 ~ (1/beta) * exp(-x/beta))
}
```

```{r}
#| code-fold: true
plot_ly(
  x = x_vec,
  y = P_x(x_vec, 3),
  type = "scatter",
  mode="line",
  name = "P(x)") %>% add_trace(
    x = 0:6,
    y = P_x(0:6, 3),
    type = "scatter",
    mode="none",
    fill = "tozeroy",
    name = "P(x) <= 6"
)
```

$$
\begin{align*}
P(X \leq 6) &= \int_0 ^6 f(x)dx \\
&= \int_0 ^6 \frac{1}{\beta} e^{-\frac{x}{\beta}}dx \\
\text{Substitute}\, u &= -\frac{x}{\beta} \\
 \frac{du}{dx} &= -\frac{1}{\beta} \rightarrow du = -\frac{1}{\beta}dx \\
P(X \leq 6) &= \int_0 ^6 \frac{1}{\beta} e^udx \\
 &= \int_0 ^6  e^u \frac{1}{\beta}dx \\
\text{Substitute}\, du &= -\frac{1}{\beta}dx \\
P(X \leq 6) &= \int_0 ^6  e^u (-du) \\
 &= -1\int_0 ^6  e^u du \\
&= - e^u \Big|_0^6 \\
P(X \leq 6) &= - e^{-\frac{x}{\beta}} \Big|_0^6 \\
\end{align*}
$$

$- e^{-\frac{x}{\beta}} \Big|_0^6 = - e^{-\frac{6}{3}} - - e^{-\frac{0}{3}} = 1 - e^{-\frac{6}{3}}$

```{r}
(-exp(-6/3)) + 1
```

When we derived the indefinite integral in order to set up our definite
integral problem, we derived another function – the cumulative density
function (cdf)

$$
F_X(x) = P(X \leq x) = \int_0^xf_X(t)dt
$$

where $f_X(t)$ is the probability density function. In our case, we can
define the cumulative density function of the exponential distribution
as follows:

```{r}
cdf_x <- function(x, beta){
  -1 * exp(-x/beta)
}
```

and confirm that $F_X(x) =   e^{-\frac{x}{\beta}} \Big|_0^6$

```{r}
1 + cdf_x(6, 3)
```

```{r}
plot_ly(
  x = x_vec,
  y = 1 + cdf_x(x_vec, 3),
  type = "scatter",
  mode="line",
  name = "F(X, Y)")
```
